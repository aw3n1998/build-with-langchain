## 一、这部剧是什么（先有故事，再有镜头）

**一句话钩子（logline）**
退役特种兵 **陈默**，隐姓埋名在女儿就读的贵族学校当保安。新来的副校长当众羞辱、要开除他，女儿被富二代同学欺凌——直到一场校车绑架案，他三分钟制服全部劫匪，军方车队封街而来，所有人才知道这个"扫地保安"是谁。

**集纲（每集一个爽点 + 一个钩子，短剧命脉）**

| 集 | 爽点/内容 | 结尾钩子 |
|---|---|---|
| 第1集 | 羞辱开局：副校长泼咖啡、女儿被嘲"你爸看大门的" | 陈默接到一个加密电话，眼神变了 |
| 第2集 | 绑架爆发：歹徒劫校车，陈默 3 分钟制服 | 远处军车封路而来 |
| 第3集 | 身份揭晓：将军当众敬礼，副校长腿软 | 幕后黑手浮现，盯上了陈默女儿 |

下面只展开**第1集**做样板（6 个分镜）。

## 二、第一集分镜脚本表（直接照这张表填进 AgentLab）

> 字段对应 AgentLab 里的：`标题 / image_prompt(出图) / motion_prompt(运镜) / narration(旁白·配音) / subtitle(屏幕字幕) / 对口型(lipsync)`。
> **对口型=是** 的镜头：人物会开口，台词写在 **旁白(narration)** 里，系统自动 TTS→S2V 对口型。
> **对口型=否** 的镜头：narration 当画外音解说，subtitle 可单独写标题/旁白字幕。

### 镜1 · 片头标题卡（对口型：否）
- **image_prompt**：暴雨夜，霓虹灯下一座贵族学校大门特写，金属校徽反光，电影质感冷色调，写实
- **motion_prompt**：缓慢推近校门，雨丝飘落，灯光摇曳
- **narration（画外音）**：他扫了三年地，没人知道，这把扫帚扫平过三个国家的战场。
- **subtitle（屏幕大字）**：王牌扫地僧
- **对口型**：否

### 镜2 · 校门口·泼咖啡羞辱（对口型：是 — 副校长说话）
- **image_prompt**：清晨校门口，一名穿保安制服的中年男人(陈默)沉静低头扫地，西装革履的傲慢副校长居高临下，写实人像，浅景深
- **motion_prompt**：镜头从扫帚摇到陈默平静的脸，再切到副校长轻蔑的表情
- **narration（=副校长台词）**：就是你？把校门口扫干净点，别让我再看见垃圾——包括你。
- **subtitle**：（留空，自动用台词）
- **对口型**：是

### 镜3 · 教室·女儿被嘲笑（对口型：是 — 同学说话）
- **image_prompt**：明亮教室，一个朴素清秀的女高中生(陈默女儿)被几个名牌打扮的同学围着嘲笑，她攥紧书包带，写实
- **motion_prompt**：镜头缓缓环绕，定格在女孩强忍的眼神
- **narration（=同学台词）**：连校服都买不起吧？你爸不就是给我们看大门的那个？
- **对口型**：是

### 镜4 · 特写·擦咖啡·旧军牌（对口型：否）
- **image_prompt**：保安陈默低头擦去脸上的咖啡，手腕一枚磨损的旧军牌特写，眼神深处一闪锋芒，冷峻写实，强对比布光
- **motion_prompt**：极缓慢推近到眼睛，背景虚化
- **narration（画外音）**：他没有抬头。有些刀，藏在鞘里太久，连自己都快忘了它有多快。
- **subtitle**：他不是不能，是不想

### 镜5 · 加密电话·神色突变（对口型：是 — 陈默一句台词）

- **motion_prompt**：镜头从手机推到陈默骤变的脸，轻微手持晃动制造紧张

## 三、在 AgentLab 里一步步怎么做（每步：做什么 / 点哪 / 填什么）

### 第 0 步 · 启动环境（每次开工先确认）
1. **确认 GPU 上的 ComfyUI 在跑**（对口型靠它）：浏览器打开
   `https://u940055-8904-812a2bab.bjb1.seetacloud.com:8443`，能看到 ComfyUI 界面=正常。
   - 如果打不开：SSH 进 4090，`tmux new-session -d -s comfy "cd /root/autodl-tmp/comfyui && /root/autodl-tmp/miniconda3/bin/python3.12 main.py --listen 0.0.0.0 --port 6006 > /root/autodl-tmp/comfyui_run.log 2>&1"` 重启。
2. **启动 AgentLab 后端**（改过 `.env` 后必须重启才生效）：
   `python -m uvicorn agent_lab.main_api:app --host 127.0.0.1 --port 8000`
3. **启动前端**：`npm --prefix frontend run dev`（默认 5173 端口）。

### 第 1 步 · 建项目
- 对 Agent 说："**建一个短剧项目，叫《王牌扫地僧》**"，或在面板入口新建。
- 对应工具：`create_video_project`。建完会进**制作面板**。

### 第 2 步 · 配角色（保证人物一致，强烈建议）
- 短剧最怕"每个镜头主角长得不一样"。先固定主角：
  对 Agent 说："**配置角色：触发词 chenmo，LoRA 用 <你的人物LoRA路径>**"。
- 对应工具：`configure_character(trigger_word, flux_lora)`。之后出图自动注入触发词，主角脸就稳了。
- 没有人物 LoRA 也能做，只是同一个人跨镜会有出入。

### 第 3 步 · 加分镜（把上面那张表灌进去）
- 逐镜对 Agent 说，或在面板加分镜，每镜填 4 个字段：
  **镜号 / 标题 / image_prompt / motion_prompt / narration / subtitle / 对口型**。
- 对应工具：`add_scene(scene_number, title, narration, image_prompt, motion_prompt, subtitle)`。
- 例（镜2）："加第2分镜，标题'泼咖啡'，旁白(台词)='就是你？……包括你。'，出图提示词='清晨校门口…'，运镜='镜头从扫帚摇到…'。"
- 旁白/提示词写完后随时能在面板「提示词」折叠区改。

### 第 4 步 · 出图 + 选图
1. **出图**：在面板每个分镜点「出图」。会弹**参数卡**（张数/步数/尺寸/seed），改好点「出图」。
   - 防抽卡建议：一次出 4 张挑最好的；竖屏短剧尺寸用 768×1024 或 832×1216。
2. **选图**：每镜出来 N 张候选，**点你最满意那张的 ✓**（select_candidate）。
   - 不满意：点「重出图」换种子再抽；或「上传图片」用你自己的图。

### 第 5 步 · 出片（这步决定每镜怎么动）—— 按镜头类型分三种

**(a) 普通运镜镜头（对口型=否，如镜1/4/6）**
- 选好图后，分镜行上**段数填 1**（或想长就填大，尾帧接续加长），点「**出视频**」。
- 看效果不够长 → 成片下方点「**➕ 再续一段**」，可反复加（看效果再决定加多少）。
- 不会写运镜？成片区点「**✨AI推荐**」，AI 据上一段末帧给你一句续段运镜，可改。

**(b) 对口型镜头（对口型=是，如镜2/3/5）—— 用 Wan2.2-S2V**
- 确认这镜**旁白里写的是台词**（人物要说的话）。
- 勾上分镜行的「**🗣 对口型**」→ 按钮变「**对口型出片**」→ 点它。
- 后台自动：台词→TTS 配音→喂 S2V→出**嘴型跟声音同步**的片（成片自带人声）。
- 第一条慢（~2-3 分钟含载模型），之后快。**已真机测通。**

**(c) 字幕和旁白不一样的镜头（如镜1/4/6 有独立 subtitle）**
- 在「提示词」里**旁白(narration)** 填画外音、**字幕(subtitle)** 填屏幕大字；两者独立。
- 留空 subtitle = 字幕沿用旁白。

### 第 6 步 · 合成整集
- 全部分镜都出完片后，对 Agent 说"**合成整集**"或面板点合成。
- 对应工具：`assemble_episode`。自动：按镜号拼接 + 旁白 TTS 配音 + 烧字幕 +（对口型镜头保留自带人声不重配）→ 一条完整短剧 mp4。

## 四、关键提示（少踩坑）

- **防抽卡**：① 一次多张挑一张，别单抽反复刷；② 先出 1 段看方向，满意再「再续一段」加长；③ 提示词不会写就点「✨AI推荐」，中/英都行（Wan 支持中文）。
- **对口型 vs 普通**：用户只需回答"这镜人物开不开口"——勾「对口型」走 S2V，不勾走普通运镜。不用碰模型名。
- **人物一致**：务必先配角色触发词/LoRA，否则主角跨镜会"变脸"。
- **竖屏**：短剧都是竖屏，出图选 768×1024 类竖屏尺寸，出片选 480×832 / 720×1280 竖屏。
